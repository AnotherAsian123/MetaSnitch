# STYLE.md

## 1. Functionality
The application should first and foremost be focused around functionality and usability. The application should not be difficult to use.

The application should also account for a responsive experience on mobile.

## 2. Overhead
Use cutting edge technology for design choices and techniques. Ensure there is not too much overhead, the application should not experience slowdowns because of unnecessary design overhead.

## 3. Stylisation
Design of the application should adopt a sleek and minimalistic approach. Design choices should be tasteful and fully responsive.

Take bold moves to ensure attention is drawn to the right areas of focus.

Resultant design should feel polished and well thought-out.

## 4. Colours

Use the following colour palette:
- #08090A - Black
- #A7A2A9 - Lilac Ash
- #F4F7F5 - Bright Snow
- #575A5E - Charcoal
- #222823 - Carbon Black

## 5. Animations
Any implementation of movement should be dynamic and smooth. Ensure there are no jarring jumps between transition.

Scrolling down pages should always be a smooth experience.
